"# IT-Governance-Case-Study-Library-Information-System" 
Overview
This project presents an evaluation of IT governance practices in a Library Information System using a simplified approach based on COBIT and ISO/IEC 27001 concepts.
The assessment focuses on identifying risks, evaluating existing controls, analyzing gaps, and providing structured recommendations to improve system reliability and security.
Objectives
Identify potential risks in the system
Evaluate existing controls
Assess IT governance maturity level
Provide improvement recommendations based on priority
